# 如何推送代码到 GitHub

你的仓库：https://github.com/Jay-6539/Primary-School-Notebook

## 方法一：使用命令行（推荐）

### 如果 Git 命令无法识别

可能的原因：
1. Git 刚安装，需要重启 Cursor/终端
2. Git 安装时没有添加到 PATH

**解决方法：**
1. **完全关闭 Cursor**
2. **重新打开 Cursor**
3. **打开新终端**（按 `Ctrl + ``）

如果还是不行，尝试：

```powershell
# 检查常见安装位置
Test-Path "C:\Program Files\Git\bin\git.exe"
Test-Path "C:\Program Files (x86)\Git\bin\git.exe"
```

如果找到了，需要添加到 PATH，或者使用完整路径。

### 执行步骤

一旦 Git 可以使用，按顺序执行：

```bash
# 1. 初始化（如果还没有）
git init

# 2. 配置用户信息（只需要做一次）
git config --global user.name "Jay-6539"
git config --global user.email "你的邮箱"

# 3. 添加所有文件
git add .

# 4. 提交
git commit -m "Initial commit: Aiden's website"

# 5. 连接远程仓库
git remote add origin https://github.com/Jay-6539/Primary-School-Notebook.git

# 6. 重命名分支
git branch -M main

# 7. 推送（需要输入用户名和 Personal Access Token）
git push -u origin main
```

## 方法二：使用 GitHub Desktop（最简单）

如果命令行有问题，推荐使用 GitHub Desktop：

### 1. 下载安装 GitHub Desktop

访问：https://desktop.github.com/
下载并安装

### 2. 登录 GitHub

打开 GitHub Desktop，用你的 GitHub 账号登录

### 3. 添加本地仓库

1. 点击 **File → Add Local Repository**
2. 点击 **Choose...**
3. 选择项目文件夹：`D:\Cursor Project\Primary School Aiden`
4. 点击 **Add repository**

### 4. 发布到 GitHub

1. 点击右上角的 **Publish repository**
2. 确保名称是：`Primary-School-Notebook`
3. 不要勾选 "Keep this code private"（保持公开，才能使用 GitHub Pages）
4. 点击 **Publish Repository**

### 5. 等待上传完成

GitHub Desktop 会自动上传所有文件。

## 方法三：直接在网页上传（临时方案）

如果上述方法都不行：

1. 访问：https://github.com/Jay-6539/Primary-School-Notebook
2. 点击 **uploading an existing file**
3. 拖拽项目文件夹中的所有文件（除了 `node_modules` 文件夹）
4. 输入提交信息：`Initial commit`
5. 点击 **Commit changes**

## 推送后启用 GitHub Pages

1. 访问：https://github.com/Jay-6539/Primary-School-Notebook/settings/pages
2. 在 **Source** 下拉菜单选择 **GitHub Actions**
3. 保存
