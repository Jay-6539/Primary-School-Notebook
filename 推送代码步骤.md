# 推送代码到 GitHub 的详细步骤

你的仓库地址：https://github.com/Jay-6539/Primary-School-Notebook

## 第一步：打开终端

在 Cursor 中：
- 按 `Ctrl + ``（反引号键，在 ESC 下面）
- 或者菜单：Terminal → New Terminal

## 第二步：检查 Git 是否可用

在终端中输入：
```bash
git --version
```

如果显示版本号（如 `git version 2.xx.x`），说明 Git 已安装，继续下一步。

如果显示错误，需要：
1. 关闭当前终端窗口
2. 重新打开 Cursor
3. 重新打开终端再试

## 第三步：初始化 Git 仓库（如果还没有）

```bash
git init
```

## 第四步：配置 Git 用户信息（首次使用需要）

```bash
git config --global user.name "Jay-6539"
git config --global user.email "你的邮箱@example.com"
```

替换 `你的邮箱@example.com` 为你在 GitHub 注册时使用的邮箱。

## 第五步：添加所有文件

```bash
git add .
```

## 第六步：提交代码

```bash
git commit -m "Initial commit: Aiden's website with English words learning"
```

## 第七步：连接到 GitHub 仓库

```bash
git remote add origin https://github.com/Jay-6539/Primary-School-Notebook.git
```

## 第八步：重命名分支为 main

```bash
git branch -M main
```

## 第九步：推送到 GitHub

```bash
git push -u origin main
```

这一步会要求你输入 GitHub 的用户名和密码（或 Personal Access Token）。

### 关于认证

如果提示输入密码：
- **不要使用 GitHub 密码**，需要使用 **Personal Access Token**

### 如何创建 Personal Access Token：

1. 登录 GitHub
2. 点击右上角头像 → **Settings**
3. 左侧菜单最下方找到 **Developer settings**
4. 点击 **Personal access tokens** → **Tokens (classic)**
5. 点击 **Generate new token** → **Generate new token (classic)**
6. 输入名称（如：Aiden Website）
7. 选择过期时间（建议选择 90 days 或 No expiration）
8. 勾选权限：**repo**（全部勾选）
9. 点击 **Generate token**
10. **复制这个 token**（只显示一次！）
11. 推送时，密码输入框粘贴这个 token

## 完成！

推送成功后，你的代码就在 GitHub 上了。然后按照之前的说明启用 GitHub Pages。
